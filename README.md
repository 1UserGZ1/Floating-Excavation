# 《Floating Excavation浮空挖掘》模组介绍
《Floating Excavation》浮空挖掘模组是一款轻量级、高效率的游戏体验优化模组，由DY_XiaoJiaZi开发，版本为1.0.0 ，适用于Minecraft Java版Forge平台，2024年发布，开源地址待发布。

## 核心理念
该模组专注解决不同环境下挖掘速度不一致的问题，提升游戏流畅性。

## 功能详解
1. **核心功能**：移除原版Minecraft两种挖掘速度惩罚。水下挖掘惩罚移除，玩家完全浸没水中（头部在水中），包括水下洞穴探索等场景，不再受5倍挖掘速度惩罚；悬空挖掘惩罚移除，玩家未站在坚实地面，如空中建筑等场景，挖掘速度不受限。
2. **智能判断系统**：模组能智能判断，完全浸没水下、水中悬空、空气中悬空及其他流体（如熔岩）中挖掘移除惩罚；站在水面未浸没、站在水底未完全浸没及正常地面挖掘保持原版情况。

## 技术特点
1. 零配置开箱即用，安装即生效。
2. 兼容性强，不修改原版核心机制，与多数模组兼容。
3. 性能友好，轻量级实现，几乎不增加系统开销。
4. 客户端/服务端同步，确保多人游戏体验一致。

## 适用版本
1. **官方支持版本**：专为Minecraft 1.20.1版本开发，Forge版本47.4.0 +，经过充分测试，推荐使用。
2. **理论兼容版本**：基于代码架构，1.20.x系列（Forge 47.x.x - 51.x.x ，可能需调整构建配置）；1.19.x系列（Forge 41.x.x - 47.x.x ，未测试，API可能变化）；1.18.x系列（Forge 40.x.x - 41.x.x ，未测试，API可能变化）。模组包含多版本构建系统，可快速适配不同版本。

## 安装指南
### 玩家安装步骤
1. **前提条件**：需Minecraft Java版、Forge 1.20.1 - 47.4.0或更高版本、Java 17或更高版本。
2. **安装流程**：下载模组JAR文件“floatingexcavation - 1.0.0 - 1.20.1.jar”；打开Minecraft游戏目录，进入“mods”文件夹（不存在则创建），将模组文件放入该文件夹，启动游戏。
3. **验证安装**：启动游戏后，在模组列表中找到“Floating Excavation”，进入游戏测试水下挖掘速度是否正常。
